Trabajando con estilos

Tenemos 3 formas diferentes de aplicar estilos a nuestra página web en HTML:

  • Estilo en línea. (Atributo style)
  • Estilo interno. (Elemento style)
  • Estilo externo. (Elemento link)

Estilos en línea (inline)

De este modo podemos hacer que toda una etiqueta muestre un estilo determinado. Por ejemplo, podemos definir un párrafo entero en color rojo y otro en color azul.

Estilo externo. (Elemento link)

Se utiliza una hoja de estilo externa para definir el estilo de muchas páginas HTML.

<head>
<link rel="stylesheet" href="style/styles.css">
</head>

En el documento styles.css tendremos definidos nuestros estilos a aplicar

Podéis observar su uso en este documento. Crearé una clase llamada clase2 sobre un elemento div y le pondremos un color de fondo azul oscuro y el texto en blanco.
