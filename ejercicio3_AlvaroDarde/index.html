El elemento q

Los navegadores normalmente insertan comillas alrededor de la cita.

El objetivo de WWF's: Build a future where people live in harmony with nature.

Abreviaturas con abbr

La etiqueta HTML define una abreviatura o un acrónimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".

The WHO was founded in 1948.

Para escribir direcciones address

La información de contacto puede ser una dirección de correo electrónico, URL, dirección física, teléfono número, identificador de redes sociales, etc.

Written by Jon Doe.
Visit us at:

Box 564, Disneyland
USA

Elemento cite

La etiqueta HTML define el título de un trabajo creativo (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.).

The Scream by Edvard Munch. Painted in 1893.

Elemento bdo: oara escribir justo al revés

BDO significa Bi-Directional Override (anulación bidireccional). La etiqueta HTML se utiliza para anular La dirección actual del texto.

Este texto será escrito de derecha a izquierda
